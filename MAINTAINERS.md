# Maintainers

This file lists the maintainers of the custos project. Maintainers are
responsible for reviewing and merging pull requests, triaging issues, and
cutting releases.

## Active maintainers

| Name | GitHub | Role |
| ---- | ------ | ---- |

## Becoming a maintainer

Consistent, high-quality contributions over time may lead to a maintainer
invitation. If you are interested, open an issue or reach out to an existing
maintainer.
